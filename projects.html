Microprocessor Connect Four

My team created a Connect Four game for our final project in our Microprocessors class. This involved programming a STM32 microcontroller, designing peripherals, and supplying power.

I was involved in all aspects of the project. I independently wrote the C code used to store the display information and detect game win conditions.

Python Image Morphing

I created a python script to morph two images for the final project of my Software Tools class.

This project involved the following:

  • Object-oriented programming
  • GUI design using the PyQt5 library
  • Image manipulation using the NumPy and SciPy libraries
  • Regular expression using the Re:Library

Sail Racing GPS

Over the summer of 2019, I designed a sailboat GPS. It utilized a STM32F0 microcontroller, Adafruit GPS, and an HD44780 LCD.

This project involved the following:

  • Microcontroller Programing - SWD & HAL library
  • PCB design - Eagle CAD
  • Data Processing - NMEA 0183
