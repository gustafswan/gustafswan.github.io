Python Image Morphing

For the final Project of my Software Tools class in the spring of 2019, I created a python script to morph two images into eachother.

This project involved the following:

  • Object Oriented Programming
  • GUI design using the PyQt5 library
  • Image manipulation using the Numpy and Scipy libraries
  • Regular expression using the RE library
